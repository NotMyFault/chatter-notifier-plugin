Each line contains a comma separated map of SCM ID to Salesforce logins.
If not set for a suspect, <SCM ID>@<Default Domain> will be used.
The 1st column is the SCM ID (for git the user name) and the second column is the Salesforce.com login.
If the login is in the default domain the @domain can be left off.
Examples:
  • joeuser,juser
  • janeuser,jane.user@gmail.com
